# ToDo App

A lightweight, responsive ToDo application with **Light/Dark mode**, task categories, notes, ICS import, and drag-and-drop ordering. Perfect for personal task management or quick organization.

---

## Features

- **Add, Edit, Delete Tasks** – quickly manage your daily to-dos.
- **Task Categories & Notes** – organize tasks with labels and additional information.
- **Drag & Drop** – reorder tasks easily.
- **Search & Filters** – filter tasks by active/done or search by keywords.
- **Tabs** – view today’s, this week’s, or all tasks.
- **Export & Print** – export tasks as JSON or CSV, or print a summary.
- **ICS Import** – import tasks/events from `.ics` files (Google Calendar compatible).
- **Light & Dark Mode** – toggle theme with persistent local storage.
- **Responsive Design** – works on desktops, tablets, and mobile devices.

---

## Usage

### Add a Task
1. Enter a task name, optional date, and category.
2. Click **Add** to save it.

### Edit a Task
1. Click **Edit** to update the task text.
2. Add notes in the detail panel.

### Mark Tasks Done
- Click **Done** to toggle completion status.

### Filter Tasks
- Use **filters** (All, Active, Done) and **tabs** (Today, Week, All) to view tasks.

### Import ICS
1. Click **Import** → select a `.ics` file.
2. Tasks/events are added automatically.

### Export / Print
- Export tasks as **JSON** or **CSV** via the buttons.
- Print a task summary using the **Print** button.

### Dark / Light Mode
- Toggle using the top-right button.
- The app remembers your preference across sessions.

## Technologies
